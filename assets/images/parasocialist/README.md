# Parasocialist Images

Place your photos here. The project page is set up to use:

- `cover.jpg` - Album artwork/cover image
- `photo1.jpg` - First photo by Asil Alptekin
- `photo2.jpg` - Second photo by Asil Alptekin

You can add more images and reference them in the project page HTML.

## Image Recommendations

- **Cover image**: Square format (1:1 ratio) works best, recommended size: 800x800px or larger
- **Photos**: Any format, recommended size: at least 1200px wide for best quality
- **File formats**: JPG or PNG
- **Optimization**: Consider compressing images before uploading to keep file sizes reasonable
